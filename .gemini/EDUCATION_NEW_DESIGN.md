## 📐 **Design Style:**

```css

```

### **Shadows:**
```css
Cards:
- Default: Soft (0 10px 40px rgba(0, 0, 0, 0.08))
- Hover: Deep (0 20px 60px rgba(0, 0, 0, 0.12))

Icons:
- Purple: 0 8px 24px rgba(139, 92, 246, 0.3)
- Blue: 0 8px 24px rgba(59, 130, 246, 0.3)

Dots:
- Ring: 0 0 0 6px rgba(59, 130, 246, 0.1)
- Shadow: 0 4px 12px rgba(59, 130, 246, 0.2)
```

### **Border Radius:**
```css
Cards: 1.6rem (16px)
Icons: 1.2rem (12px)
Tags: 6px
Status: 5rem (pill shape)
```

## ✨ **Hover Effects:**

### **Card Hover:**
```
1. Slides 8px to the right (or left for even items)
2. Left gradient border appears (blue → purple)
3. Shadow intensifies
4. Icon scales 1.08x and rotates -5deg
Duration: 0.4s cubic-bezier
```

### **Dot Hover:**
```
1. Scales to 1.4x
2. Ring expands to 10px
3. Shadow intensifies
Duration: 0.3s ease
```

### **Tag Hover:**
```
1. Background changes to blue
2. Text turns white
3. Lifts up 2px
4. Blue shadow appears
Duration: 0.2s ease
```
